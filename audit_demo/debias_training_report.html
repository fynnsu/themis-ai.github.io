Overall Metrics

The following plots depict the overall performance metrics of the model in the default state (no debiasing) and after utilizing the debiased training module. You can utilize these plots to calculate the financial benefit of debiased training.

Each metric is calculated as follows:  

\[ Accuracy\ =\ \frac{correct\ prediction}{all\ predictions}\]

\[ recall\ =\ \frac{True\ Positive}{True\ Positive\ +\ False Negative}\]

\[ precision\ =\ \frac{True\ Positive}{True\ Positive\ +\ False Positive}\]

\[ f1\ =\ 2\ .\ \frac{precision\ .\ recall}{precision\ +\ recall}\]
