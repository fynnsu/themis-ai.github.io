Overall Metrics

The following plots depict the overall performance metrics of the baseline (uniform) model and debiased module. You can utilize these plots to calculate the financial benefit of debiased training.

Each metric is calculated as follows:  

\[ Accuracy\ =\ \frac{correct\ prediction}{all\ predictions}\]

\[ recall\ =\ \frac{True\ Positive}{True\ Positive\ +\ False Negative}\]

\[ precision\ =\ \frac{True\ Positive}{True\ Positive\ +\ False Positive}\]

\[ f1\ =\ 2\ .\ \frac{precision\ .\ recall}{precision\ +\ recall}\]

Label and Feature Metrics

The following plots compare the performance of each label before (uniform model) and after debiasing (debiased model) was enforced during training. As you can observe the debiased training was able to improve the performance on the underrepresented classes with no (or negligible) compromise on overrepresented classes.

Train History

In the following plots, we observe debiased model loss is higher than the baseline (uniform) model during training and lower during testing. This is because our debised training algorithm always finds the challenging data samples and focuses the training time on the areas the model hasn't learned yet. This results in a higher train loss and a better performing model (i.e. lower test loss).
